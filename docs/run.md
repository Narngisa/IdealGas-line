## Getting Started 📗

1. Create vitual environment
```
python -m venv venv
```
2. Run vitual environment
> [!IMPORTANT]
> WINDOW
> ```
> /venv/Scripts/activate
> ```
> MacOS / Linux
> ```
> source venv/bin/activate
> ```

3. Install library
```
pip install -r requirements.txt
```
4. Run It!
```
python main.py
```
