## ❓How to run❓

> [!NOTE]
> Python version 3.9 or higher

- Git 👇
```
git clone https://github.com/Narngisalnw/IdealGas-line.git
```
#

- Library Python 🔰
```
pip install -r requirements.txt
```

- Visual Studio Code or Others program 💻
```python
python main.py
```

#

- Command Prompt 📗
```
cd idealgas-line
```
```
.\env\Scripts\activate
```
```
python main.py
```

- PowerShell 📘
```
cd idealgas-line
```
```
env\Scripts\activate
```
```
python main.py
```
